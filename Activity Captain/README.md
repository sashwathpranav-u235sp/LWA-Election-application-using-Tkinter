

The Activity Captain Election application is intended to collect and process votes from students to elect Lightworkers Academy's Activity Captain for the 2022/2023 academic year. The application is built using Python and Tkinter with vote collection and processing handled by two separate python files (main.py and results.py respectively). 

main.py is responsible for generating the GUI for the application and collecting the votes cast by students. The data is then stored in info.csv which is processed by results.py to give the results for a polling session and determine the winner of the election.
